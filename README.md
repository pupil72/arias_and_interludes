# arias_and_interludes
"Arias and Interludes is a piano suite I assembled that uses an expressly built-for Android application as an interface between whatever sound is produced in, on or with the piano and a vibration speaker: the sound is captured by the microphone of the Android device, then it is processed by the application and eventually is sent via Bluetooth to the speaker sitting on the piano soundboard, through which it gets amplified."

This app is only for Android and needs two libraries to be able to work: OpenCV (https://opencv.org/android/) and TarsosDSP (https://github.com/JorenSix/TarsosDSP)

It includes the code of "follow me" a former, simpler version of the Optical Theremin published in Google Play (https://play.google.com/store/apps/details?id=marcoalunno.optical_theremin&hl=it&gl=US).
